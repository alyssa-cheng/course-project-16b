Each voter ranks their choices in order of first, second, third, and so on. Then, the voting system performs the following steps until there is only one option left.

  • The system counts how many first place votes each choice (from the remaining choices possible) received.
  • Let x and y denote the two choices which got the most first place votes. The system counts how many voters rank x higher than y. The system also counts how many voters rank y higher than x. Say that x is preferred by more voters than y.
  • The system removes y as an option from the set of all options.

The voting system ranks the choices in the order of when they were removed, with the winner being the first option that was removed.
